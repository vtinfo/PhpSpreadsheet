BESSELY.

public static BESSELY( $x : mixed , $ord : mixed ) : float|string

Returns the Bessel function, which is also called the Weber function or the Neumann function.

Excel Function: BESSELY(x,ord)

Parameters
$x : mixed

A float value at which to evaluate the function. If x is nonnumeric, BESSELY returns the #VALUE! error value.

$ord : mixed

The integer order of the Bessel function. If ord is not an integer, it is truncated. If $ord is nonnumeric, BESSELY returns the #VALUE! error value. If $ord < 0, BESSELY returns the #NUM! error value.

Return values
float|string —

Result, or a string containing an error
