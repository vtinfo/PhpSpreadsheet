public static format( $number : mixed [, $precision : mixed = 2 ] ) : string

This function converts a number to text using currency format, with the decimals rounded to the specified place. The format used is $#,##0.00_);($#,##0.00)..

Parameters
$number : mixed

The value to format

$precision : mixed = 2

The number of digits to display to the right of the decimal point (as an integer). If precision is negative, number is rounded to the left of the decimal point. If you omit precision, it is assumed to be 2

Return values
string
